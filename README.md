# FastAPI Todo API

Simple API to manage tasks
Built with Python and FastAPI

## Tech Stack
- Python 3
- FastAPI
- SQLite

## How to Run

1. Install dependencies
```bash
pip install -r requirements.txt
```

2. Run the server
```bash
uvicorn main:app --reload
```

3. Open Swagger UI
http://127.0.0.1:8000/docs
